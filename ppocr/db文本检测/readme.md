# 最简单模型版

运行代码需要安装`paddlepaddle` 和`PaddleOCR`



`cpu`版paddle 安装

```bash
python -m pip install paddlepaddle==2.2.2 -i https://mirror.baidu.com/pypi/simple
```

安装`PaddleOCR`

```
pip install paddleocr
```
